# Дизайн и компоненты

## Дизайн-система

**Стиль:** Современный минимализм  
**Цвета:** Tailwind CSS с кастомной темой  
**Компоненты:** Shadcn/UI + Radix UI

## Основные компоненты

**UI Components** (`src/components/ui/`):

- Button, Card, Dialog, Input, Select, Table, Badge

**Бизнес-компоненты:**

- `proposal-form-v3.tsx` - форма предложений
  - Фиксированные ширины колонок
  - Автоматический расчет площади в названиях
  - Поддержка переносов в описаниях товаров
- `product-configurator-v2.tsx` - конфигуратор V2
  - Отдельный блок "Размеры" в начале
  - Кнопка Conferma в левой панели
  - Порядок параметров: Ширина → Высота
- `parameters-configuration.tsx` - конфигурация параметров
  - Визуальное выделение блока размеров
  - Защита системных параметров от удаления
- `parameters-manager.tsx` - управление параметрами
  - Бейдж "Системный" для защищенных параметров
- `vat-rates-manager.tsx` - управление ставками НДС
  - Защита системных ставок от удаления
- `document-statuses-manager.tsx` - управление статусами документов
  - Таблица с типами документов
  - Цветные бейджи
  - Модальное окно редактирования
- `clients-sticker-v2.tsx` - список клиентов
- Модальные формы: client, supplier, partner, installer

**Утилиты:**

- `src/lib/client-input-parser.ts` - умный парсер клиентов
  - Автоматическое распознавание телефона, email, имени
  - Валидация итальянских номеров
  - Автоисправление регистра

## Языки

Поддержка: Русский / Итальянский  
Переключение: `LanguageContext`  
Локализация:

- Булевые параметры: RU="Да/Нет", IT="Sì/No"
- Заголовки блоков: RU="Размеры", IT="Dimensioni"
- Системные метки: RU="Системный", IT="Sistema"
